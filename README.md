# LBD Portfolio - 个人作品导航页

> 用 AI 探索创造的无限可能

## 简介

个人作品导航页面，展示 AI 辅助创作的学习历程和实验项目。

## 特性

- **全屏滚动** - 垂直方向平滑过渡
- **响应式布局** - 优化适配桌面、平板、移动设备
- **主题切换** - 支持亮色/暗色模式
- **页面记忆** - 刷新后自动恢复浏览位置

## 技术栈

- HTML5 + CSS3
- 原生 JavaScript
- LocalStorage
