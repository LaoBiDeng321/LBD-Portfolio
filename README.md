# LBD Portfolio - 个人作品导航页

> 用 AI 探索创造的无限可能

## 简介

这是一个工业级标准的个人作品导航页面，采用全屏滚动设计，展示 AI 辅助创作的学习历程和实验项目。

## 特性

- **全屏滚动** - 垂直方向平滑过渡，支持鼠标滚轮、触摸滑动、键盘导航
- **响应式设计** - 适配桌面、平板、移动设备
- **主题切换** - 支持亮色/暗色模式，自动保存用户偏好
- **页面记忆** - 刷新后自动恢复到上次浏览的位置
- **内部滚动** - 作品展示区域支持独立滚动

## 技术栈

- HTML5 + CSS3 (Glassmorphism 设计风格)
- 原生 JavaScript
- SVG 图标
- LocalStorage 本地存储

## 页面结构

1. **首页** - 个人简介与导航入口
2. **关于我** - 个人介绍与理念
3. **作品展示** - 6个 AI 辅助创作的项目
4. **技能专长** - 正在学习的 AI 工具与开发技能
5. **联系方式** - 邮箱、社交链接与联系表单
